Project Description
AvielGabaiProject is a portfolio website that showcases various HTML, CSS, and JavaScript projects developed by Aviel Gabai. This project highlights the skills, creativity, and technical expertise in front-end web development. Additionally, it provides an overview of Aviel Gabai's background, skills, and contact information.

Responsive Design: Ensures optimal viewing experience across different devices.
About Me Section: Information about Aviel Gabai, including skills, background, and interests.
Contact Form: Allows visitors to send messages directly.
